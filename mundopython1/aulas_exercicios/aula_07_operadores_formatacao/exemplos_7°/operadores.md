# Operadores Aritmédicos do Python.

# Adição: +                Ex: 5 + 2 == 7
# Subtração: -             Ex: 5 - 2 == 3
# Multiplicação: *         Ex: 5 * 2 == 10
# Divisão: /               Ex: 5 / 2 == 2.5
# Divisão inteira: //      Ex: 5 // 2 == 2
# Potência: **             Ex: 5 ** 2 == 25
# Resto da divisão: %      Ex: 5 % 2 == 1
