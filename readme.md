## 📂 Organização das Aulas

### [Mundo 1: Fundamentos](./mundopython1/aulas_exercicios/)
*   **Aula 04:** Primeiros passos e saída de dados.
*   **Aula 06:** Tipos primitivos e manipulação de dados.
*   **Aula 07:** Operadores aritméticos e formatação.
*   **Aula 08:** Utilizando módulos e bibliotecas.
*   **Aula 09:** Manipulando cadeias de texto (Strings).
*   **Aula 10:** Condições simples (`if`/`else`).

### [Mundo 2: Estruturas de Controle](./mundopython2/aulas_exercicios/)
*   **Aula 12:** Condições aninhadas (`elif`).

*   **Aula 14:** Estrutura de repetição `while`.
*   **Aula 15:** Interrompendo repetições `while` (break).
